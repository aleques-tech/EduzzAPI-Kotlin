# EduzzApi-Kotlin

A Kotlin (Java Compatible) Library for acessing the [Eduzz API](https://api2.eduzz.com/).

On Gradle/Kts, use:

    repositories {
        ...
        maven("https://jitpack.io")
    }
    
    dependencies {
        ...
        implementation("com.github.aleques-tech:EduzzAPI-Kotlin:0.1.2")
    }
